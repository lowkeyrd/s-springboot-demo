# 阿里云函数计算 Java 8 案例

只需几步就可以快速在阿里云函数计算服务上体验 Java 8 ：

- 初始化项目：`s init start-fc-event-java8`
- 进入项目：`cd start-fc-event-java8`
- 部署项目：`s deploy`
- 触发项目：`s invoke remote`

即可实现`Java 8`案例的初始化、部署整个流程。

> 本应用仓库地址：https://github.com/devsapp/start-fc
